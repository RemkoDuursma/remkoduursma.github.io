Description

Package includes 3 raw data files from an experiment examining the dynamic physiology of sun and shade leaves. This physiology was related to light gradient driven resource distribution within the tree canopy. This data is associated with the paper: 'Coupled response of stomatal and mesophyll conductance to light enhances photosynthesis of shade leaves under sunflecks' in Plant, Cell & Environment. Individual file descriptions can be found in the embedded README.html.

Code WTC
Description The Whole Tree Chambers (WTC) facility was installed in year 2006 and is located at the Hawkesbury Forest Experiment at the University of Western Sydney on Southee Road, Richmond. They have utilised twelve (12) whole-tree chambers that house trees up to 10 m tall and can accurately measure whole tree CO2 uptake and water loss. Each individual chamber has the ability to control CO2 concentration, air temperature and vapour pressure deficit (VPD).

Basic Information

Type RAW
Size 11KB
File format text/plain
Description Leaf trait data for sun and shade leaves grown in whole tree chambers with ambient (n=6) and +3C warming treatment (n=6). Data are for the paper: Courtney E. Campany, Mark G. Tjoelker, Susanne von Caemmerer, Remko A. Duursma. in press. Coupled response of stomatal and mesophyll conductance to light enhances photosynthesis of shade leaves under sunflecks. Plant, Cell and Environment. This experiment includes a drought treatment, which is excluded from this data set. This reduces the sample size from n=6 to n=3 over the last 3 months of the dataset for each temperature treatment. This file includes leaf-level trait estimates for sun and shade leaves. Variables: Year : 2013 or 2014 Month : month of measurements (campaigns were several days each) campaign: campaign number (1-6) chamber : individual whole tree chamber leaf : leaf type (sun or shade) temp : ambient or elevated (+3C) pre_wp : predawn leaf water potential (MPa) mid_wp : midday leaf water potential (MPa) Kl : leaf specific hydraulic conductance (mmol m-2 s-1 MPa-1) delta13c : leaf δ13C (‰) n_perc : mass-based leaf nitrogen (%) lma : leaf mass per area (g m-2) leafN_area : area-based leaf nitrogen content (gN m-2)
